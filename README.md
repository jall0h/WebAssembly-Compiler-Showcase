# A-FUN Compiler Showcase

## Description
Web playground for a compiler, translating A-FUN functional programming language to WebAssembly Text Format (WAT) executed in the browser.

## Installation
Prerequisites
- Node
- pnpm

1. Clone repository
```sh
git clone <repo-link>
```

2. Install packages

  ```sh
pnpm install
```
3. Run development environment

```sh
pnpm run dev
```

## Tech Stack
- React with Vite
- TypeScript
- WebAssembly
